# Job Sheet Validator

A Streamlit prototype designed to reduce invoicing delays caused by missing or incorrect job sheet information.

## Purpose

This tool simulates a technician job sheet validation workflow.

It checks:
- required PO numbers
- PO format
- contract type compatibility
- missing technician notes
- invoice risk level

## Tech Stack

- Python
- Streamlit
- Pandas

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
